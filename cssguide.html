CSS Explained


Hover over this to find out what CSS entails !!!

CSS stands for Cascading Style Sheets. CSS is a standard style sheet language used for describing the presentation (i.e. the layout and formatting) of the web pages.

CSS is performed using a wide array of selectors and identifiers to ensure maximum customization

Examples of how CSS can be used in web design

Using the background image selector we changed the background of our website into a colour gradient
We also used css to align the images and videos on our website

We can even get very specific in our customization and use our creativity and imagination and do all sorts of things 🤪
